# SCT_WD_4_ToDo_Web_App
A responsive To-Do Web Application built with HTML, CSS, and JavaScript featuring task management, date &amp; time scheduling, completion tracking, edit/delete options, filters, and localStorage support.# 📝 SkillCraft Task 04 – To-Do Web Application

This project is a **responsive To-Do Web Application** developed using **HTML, CSS, and JavaScript**. It helps users manage daily tasks efficiently with a clean and user-friendly interface.

## 🚀 Features
- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed
- Set date and time for tasks
- Filter tasks by **All / Pending / Completed**
- View task statistics
- Store tasks in **localStorage** so data remains saved after refreshing the page
- Responsive design for desktop and mobile devices

## 🛠️ Technologies Used
- **HTML5**
- **CSS3**
- **JavaScript**

## 📌 Project Purpose
The main goal of this project is to practice **DOM manipulation, event handling, localStorage, and responsive web design** by building a useful real-world task management application.

## 📷 Output
The To-Do Web App allows users to:
- organize daily tasks
- track completed and pending work
- schedule tasks with date and time
- manage tasks easily with edit and delete options
